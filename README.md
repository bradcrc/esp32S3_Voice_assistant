# esp32S3_Voice_assistant
Voice Assistant with local microwakeword

Source for a simple Home Assistant voice assistant built with ESP32-S3-WROOM-1 chip, microphone, and speaker.

Just putting source up here since it was far too difficult for me to find working examples for this config.  I don't plan on updating, so feel free to fork and whatever.  This is just to help others get started.
